Overview

TrashScan addresses the critical limitation of traditional urban litter monitoring that relies on manual inspection methods lacking scalability and continuous coverage. By leveraging existing CCTV infrastructure already installed for security purposes, the system transforms reactive municipal waste management into proactive, data-driven environmental monitoring.

The system integrates YOLOv8 with Slicing-Aided Hyper Inference (SAHI) to tackle small object detection challenges in high-resolution surveillance footage. Through systematic optimization including genetic algorithm-based hyperparameter tuning, the model achieved superior performance with mAP@50 of 0.94 and 22.8ms GPU inference time. SAHI integration enhanced small object detection by 30.7%, while genetic algorithm optimization improved overall performance by 10.8%.

Successfully deployed in Barangay 118 Zone 9, Tondo, Manila, TrashScan implements round-robin scheduling for multi-feed processing and employs a modified Clean Environment Index for quantitative density assessment, providing municipal authorities with actionable insights through automated notifications and comprehensive analytics.

Key Features

YOLOv8 + SAHI Integration

Advanced computer vision combining YOLOv8's real-time detection with Slicing-Aided Hyper Inference for 30.7% improvement in small object detection, crucial for identifying litter at typical surveillance camera mounting heights.

Multi-Feed Round-Robin Processing

Efficient scheduling mechanism cycles through multiple CCTV feeds at customizable intervals, enabling balanced resource allocation without requiring prohibitive hardware investments for municipal deployments.

Modified Clean Environment Index

Spatially-weighted density calculation that quantifies litter impact through geometric analysis, providing objective measurements (LOW, MEDIUM, HIGH) translated into clear action thresholds for municipal personnel.

Automated Alert System

Real-time notifications via Twilio SMS and email when litter density exceeds predefined thresholds, enabling immediate response from waste management teams with location-specific alerts.

Comprehensive Analytics Dashboard

Interactive visualizations with D3.js showing detection logs, temporal trend analysis, ranking systems, and heat maps. Map-based interface with color-coded polygons indicates current litter density levels by street segment.

Optimized Performance

Genetic algorithm-based hyperparameter optimization achieved 10.8% performance improvement. Minimal hardware requirements (2GB GPU VRAM, 4GB CPU RAM) make it suitable for edge deployment in resource-constrained barangay infrastructure.
