Overview

SARI is a comprehensive website platform designed to streamline sales, stock management, and transaction logging for small businesses and MSMEs. Built with the unique needs of retail operations in mind, particularly sari-sari stores, SARI offers an intuitive and feature-rich experience that helps store owners take control of their inventory and financial records seamlessly.

As the team leader, I guided the development of this full-stack business management system using a PHP-MySQL architecture with a Bootstrap frontend. The system features robust role-based authentication, secure password hashing, and comprehensive validation and sanitation procedures to ensure data integrity and security throughout all operations.

The platform was successfully deployed to production with real-time analytics dashboards powered by Chart.js and Bootstrap 5, delivering daily, monthly, and yearly sales insights through responsive design and jQuery-powered dynamic updates. SARI represents a complete business intelligence solution tailored specifically for the Philippine MSME market.

Key Features

Secure Authentication System

Built with PHP password hashing and MySQL prepared statements, implementing role-based access control for admin/cashier privileges with session management. Advanced password hashing techniques provide protection against unauthorized access and data breaches.

Complete POS Transaction Engine

Streamlines customer checkout with product lookup, real-time stock adjustments, discount calculations, payment validation, and automated receipt generation. Features an AJAX shopping cart and purchase confirmation workflows for smooth transactions.

Real-Time Analytics Dashboard

Provides a centralized hub for monitoring key operational and financial metrics using Chart.js and Bootstrap 5. Displays daily, monthly, and annual earnings with customizable financial projection graphs for forecasting and planning.

Comprehensive Inventory Management

Provides robust tools for inventory management through proper database operations. Supports adding new products, tracking stock movements (in and out), updating product details, and removing items with comprehensive information including product codes, quantities, and prices.

Role-Based User Management

Allows tight control over user access and permissions. View total user accounts, assigned roles, and account creation timestamps. Administrators can reset passwords, remove user access, add cashier accounts, and view transaction history for each user.

Complete Audit Trail System

Captures a comprehensive record of user actions including login/logout events, inventory changes, and sales transactions. Maintains detailed sales reports analyzing financial performance and ensuring transparency for regulatory compliance.
